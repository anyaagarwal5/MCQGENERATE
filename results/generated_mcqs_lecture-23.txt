## MCQ 1
Question: What is Cross-Site Request Forgery (CSRF) also known as?
A) XSS
B) SQL Injection
C) Sea Surf or Session Riding
D) Phishing
Correct Answer: C

## MCQ 2
Question: Which of the following is NOT a potential consequence of a successful CSRF attack?
A) Unauthorized fund transfers
B) Changed passwords
C) Data theft (stolen session cookies)
D) Denial of Service (DoS)
Correct Answer: D

## MCQ 3
Question: How does CSRF exploit user authentication?
A) By bypassing the login process
B) By cracking the user's password
C) By using the user's active session
D) By installing malware on the user's computer
Correct Answer: C

## MCQ 4
Question: Which of these is a primary method for preventing CSRF attacks?
A) Using strong passwords
B) Installing a firewall
C) Implementing anti-CSRF tokens
D) Regularly updating software
Correct Answer: C


## MCQ 5
Question: What is the function of the SameSite cookie attribute?
A) To encrypt cookies
B) To restrict cookie usage to first-party contexts
C) To store user credentials securely
D) To prevent XSS attacks
Correct Answer: B
